How Split & Merge Works
  1. Upload one or multiple PDF files using the upload area.
  2. Click on a PDF file to choose whether you want to split it or include it for merging.
  3. Select specific pages or page ranges you want to keep.
  4. Rearrange files if needed, then merge the selected pages into one PDF.
  5. Download your newly created PDF file instantly.
